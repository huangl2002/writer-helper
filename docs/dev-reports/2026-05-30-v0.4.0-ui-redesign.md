# 开发报告：v0.4.0 UI 灵动化重设计

## 日期：2026-05-30

## 版本：v0.4.0 (minor bump — UI breaking visual change)

## 概述

全面重构视觉系统，从"能用"升级到"好看"。新增全局动画系统、重构色彩变量体系、重设计首页、优化编辑器/对话/弹窗的微交互。

---

## 新增

### 1. 全局动画系统 (`src/styles/animations.css`)

| 类名 | 效果 |
|------|------|
| `animate-fade-in` | 淡入 |
| `animate-fade-in-up` | 从下方滑入淡入 |
| `animate-slide-left` | 从左滑入 |
| `animate-scale-in` | 缩放入场 |
| `animate-stagger` | 子元素依次入场（10 级延迟） |
| `hover-lift` | 悬浮上浮 + 阴影增强 |
| `paper-elevation` | 纸张多层阴影 |
| `msg-enter` | 消息气泡入场 |
| `shimmer` | 骨架屏加载效果 |
| `pulse-ring` | 脉冲光晕（状态指示） |
| `gradient-text` | 渐变文字 |
| `cursor-blink` | 光标闪烁 |

### 2. 主题系统升级 (`themes.css`)

- **色彩变量**：从 6 个扩展到 15 个（新增 hover、muted、accent-light、accent-soft、success/warning/danger、glow）
- **阴影系统**：sm/md/lg/xl 四级阴影
- **圆角系统**：sm/md/lg/xl 四级圆角
- **字体**：引入 Inter + 抗锯齿渲染
- **滚动条**：自定义细滚动条样式
- **选择色**：使用主题 accent 色
- 浅色主题 accent 改为靛蓝（#4f46e5），更现代
- 暗色主题 accent 改为柔紫（#818cf8）

### 3. 首页重设计 (`HomePage.tsx`)

- **Hero 区域**：状态指示点 + 渐变标题
- **统计卡片**：悬浮上浮 + 更大字号 + 鼓励文案
- **CTA 按钮**：渐变背景 + 发光阴影 + 按压反馈
- **功能卡片**：带图标容器 + 顶部渐变条（hover 显示）+ 上浮动画
- **页脚**：胶囊形版本信息栏

### 4. 组件级增强

- **编辑器**：纸张改用 paper-elevation 多层阴影
- **工具栏**：按钮 hover 显示边框、微动画
- **状态栏**：毛玻璃效果
- **AI 对话**：气泡圆角 + 入场动画 + 左右区分更明显
- **搜索框**：毛玻璃背景 + 缩放入场
- **弹窗**：毛玻璃背景 + 缩放入场
- **AI 配置**：页面淡入、表单缩放入场
- **辅助面板**：统计卡片渐变背景

### 5. 辅助面板

- 今日统计卡片使用 accent 色渐变背景
