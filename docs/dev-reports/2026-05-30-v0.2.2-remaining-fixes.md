# 开发报告：v0.2.2 遗留问题修复

## 日期：2026-05-30

## 版本：v0.2.2

## 概述

基于测试报告中的遗留问题，修复了 AI 配置错误提示、测试连接、打字机滚动、趋势图表、角色头像、对话持久化、作品维度统计 7 个问题。

---

## 修复清单

### 1. AI 配置错误提示修复 (B43)

**问题：** 配置 AI 后仍提示"未配置 AI 服务"。原因是：(1) 解密失败时错误被吞掉，(2) 未设默认时直接返回 null。

**修复：**
- AiChat 加载流程重构：先查默认配置 → 查找是否有任何配置 → 显示具体错误
- 三种错误消息：解密失败 / 未设默认 / 加载失败
- 新增"重新加载"按钮，手动重试
- AiSettings 新增"测试连接"按钮，发送简短请求验证 API 可用

### 2. 打字机滚动模式 (B7)

**问题：** 写多行后光标走到屏幕底部，需手动滚动。

**修复：**
- 新增"打字机"按钮（工具栏旁），开启后光标保持在视口上方 1/3
- 在编辑器 onUpdate 中检测光标位置，自动滚动
- 持久化到 localStorage
- 修改文件：`WritingEditor.tsx`

### 3. 30 天码字趋势图 (B18)

**问题：** 统计页面只有年度热力图，缺少日常趋势。

**修复：**
- 新增 `get_recent_stats` 后端命令（`stats.rs`），返回最近 N 天每日字数
- 注册 Tauri 命令（`lib.rs`）
- 前端 Canvas API 绘制柱状图：深色柱为有码字日，淡色为无码字日
- 自适应 DPR，网格线，每 7 天标注日期
- 修改文件：`stats.rs`, `lib.rs`, `db.ts`, `StatsPanel.tsx`

### 4. 角色头像 (B21)

**问题：** 角色卡片只有文字，缺少视觉识别。

**修复：**
- CharacterForm 新增头像上传区（圆形虚线框），支持本地图片选择
- 头像以 base64 data URL 存储于 localStorage
- CharacterCard 显示头像或默认占位符
- 支持移除头像
- 修改文件：`CharacterForm.tsx`, `CharacterCard.tsx`

### 5. AI 对话持久化 (B41)

**问题：** 切换页面后对话历史丢失。

**修复：**
- messages 初始值从 localStorage 恢复
- 每次 messages 变化时写入（最多保留 100 条）
- 新增"清除"按钮手动删除历史
- 修改文件：`AiChat.tsx`

### 6. 作品维度统计 (B19)

**问题：** 统计页面无法按作品筛选。

**修复：**
- 统计页面新增作品选择器（下拉框）
- 全部作品模式下显示每部作品卡片（字数 + 章节数）
- 点击卡片可筛选单作品视图
- 纯前端实现，利用 store 中的 chapters 数据
- 修改文件：`StatsPanel.tsx`

### 7. 辅助面板实时刷新 (B15)

**问题：** 写完后辅助面板字数不变。

**修复：** HelperPanel 的 `useEffect` 依赖从 `[]` 改为 `[chapters.length]`，每次数据变更时刷新今日统计。
