# AI Writer Helper 用户手册

## 快速入门

1. 启动应用 → 点击「+ 新建作品」创建你的第一部小说
2. 在侧栏中点击「+ 卷」「+ 章」建立结构
3. 选中章节开始写作
4. 点击「AI 配置」添加 API（如 DeepSeek），开始使用 AI 辅助

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 手动保存 |
| `Ctrl+F` | 查找 |
| `Ctrl+H` | 查找替换（展开替换栏） |
| `Ctrl+Shift+F` | 全局搜索（跨所有章节） |
| `Ctrl+B` | 加粗 |
| `Ctrl+I` | 斜体 |
| `Ctrl+U` | 下划线 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Enter` | 查找下一个 / AI 发送 |
| `Shift+Enter` | 查找上一个 |
| `Esc` | 关闭弹窗 / 关闭查找栏 |

## 功能说明

### 码字台
- **工具栏**：编辑（撤销/重做/查找）/ 格式（加粗/斜体/下划线/删除线/高亮）/ 段落 / 列表 / 对齐 / 工具（一键排版）/ 插入
- **控制按钮**：侧栏 / 辅栏 / 专注模式 / 打字机模式
- **一键排版**：点击 📐 自动段间空行、全角标点转换
- **AI 选区工具栏**：选中文字后自动出现续写/润色/扩写/缩写按钮
- **查找替换**：Ctrl+F 查找，Ctrl+H 替换，显示匹配计数，支持全部替换
- **打字机模式**：光标保持在屏幕上部 1/3，内容自动上移
- **崩溃恢复**：意外退出后重开章节，提示恢复未保存内容

### 侧栏
- 作品 → 卷 → 章节三级树
- 上移/下移排序（hover 显示 ↑↓）
- 右键菜单：重命名 / 状态变更 / 移至其他卷 / 删除
- 搜索过滤

### AI 助手
- **5 种模式**：续写 / 润色 / 扩写 / 缩写 / 自由对话
- **流式输出**：AI 逐字输出，无需等待
- **选区集成**：编辑器内选中文字→浮动工具栏→一键生成→插入编辑
- **对话持久化**：翻页不丢对话记录
- **配置方式**：AI 配置页添加 OpenAI 兼容 API → 点击 ⭐ 设为默认

### 角色管理
- 姓名 / 性别 / 别名 / 外貌 / 性格 / 背景
- **头像上传**：点击圆形头像框选择本地图片
- 角色关系管理（朋友/敌人/恋人/师徒/家人/其他）

### 大纲规划
- 无限层级树，节点类型：卷 / 章 / 情节 / 场景
- 上移/下移排序
- 关联已写章节
- 完成标记

### 数据统计
- **30 天趋势柱状图**：深色=有码字，浅色=未码字
- **作品维度筛选**：按单作品查看统计
- 年度热力图（GitHub 风格 5 级色阶）

### 写作目标
- 每日目标字数 + 实时进度条
- 番茄钟：25 分钟专注 + 5 分钟休息，自动循环
- 桌面通知提醒

### 历史版本
- 每 25 次保存自动快照
- 手动创建快照
- **版本对比**：选中两个快照，红色/绿色高亮差异
- 恢复历史版本

### 导入导出
- 导出：单章 TXT/MD / 全书 TXT/MD / 大纲 MD / 角色 JSON
- 导入：TXT（按章节标记拆分）/ MD（按标题拆分）/ 文件夹批量导入

### 主题
- 浅色 / 暗色 / 护眼（米黄）三套主题
- 一键切换，自动保存偏好
